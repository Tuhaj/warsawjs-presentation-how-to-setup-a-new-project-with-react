Project Assumptions

  • make it progressive
  • support for modern browsers and IE11
  • we should be able to move components between projects easily

How to start with create-react-app?

  • create-react-app warsawjs-app
  • cd warsawjs-app
  • npm start

What's there in the initial config?

  • Hidden Webpack config in react-scripts
  • Jest as testing library with autorun!
  • Support for PWA
  • No Configuration Required

Not everyone likes this idea

Example of a test written in Jest

            
                import sum from './sum';

                it('sums numbers', () => {
                  expect(sum(1, 2)).toEqual(3);
                  expect(sum(2, 2)).toEqual(4);
                });

What is missing, but we hope to have in the future?

  • Hot Module Reloader

Let's try this config!

  • some rigorous eslint perset
  • styled-components (when eject remove PostCSS, css-loader etc.)
  • deep-freeze for testing immutability
  • lodash
  • No generators. If you like them use e.g. react-boilerplate

Caveats

  • Airbnb eslint perset won't allow you returning an assignment
  • async / await is supported, but does it give polyfills to the build?
